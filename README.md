Bad Apple by Processing for NITGC  
====
ProcessingでBad Appleを再生するだけ  
プログラミング基礎の課題**提出用**
## 前提
[Processing Sound library](https://processing.org/reference/libraries/sound/)  
**最低** : 4GB弱の利用可能メモリ  
**推奨** : 5GBの利用可能メモリ  
**※メモリが少ない場合、OutOfMemoryError発生の可能性有**
